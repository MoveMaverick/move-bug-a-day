## **Insert Range Check In Utils**

**Vulnerability Pattern:** Input Validation

**Severity:** Low

**Ecosystem:** Sui

**Protocol:** Maven

**Auditor:** OtterSec

**Report:**

[https://ottersec.notion.site/Sampled-Public-Audit-Reports-a296e98838aa4fdb8f3b192663400772](https://www.notion.so/Sampled-Public-Audit-Reports-a296e98838aa4fdb8f3b192663400772?pvs=21)

**Report Date:** Apr 2023

**Description:**

In utils.move, vector_slice should return the subslice of the vector, starting at the start index
and ending at the end index. However, there is no check to ensure that end is higher than start; in this case, the function returns an empty vector.

**Recommendation:**

Ensure that the function aborts if end < start rather than returning an empty array.

**Coding/Auditing Tip:** Ensure function parameters have properly designed checks.
